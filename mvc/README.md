MVC
===

### Example App add-on structure for Yii2

```
yii2-app/
    ├── components/     Yii2 customized component
    ├── forms/          Form
    ├── heplers/        Helper
    ├── presenters/     Presenter
    ├── repositories/   Repository
    ├── services/       Service
    └── widgets/        Yii2 Widget
```
